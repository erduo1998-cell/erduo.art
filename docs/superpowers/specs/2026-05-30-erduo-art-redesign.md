# erduo.art 网站重设计 v3 — 设计文档

> 日期：2026-05-30
> 从 v2 全屏翻页改造为纵向长滚动品牌官网

## 一、设计目标

在保持 v2 电影感视觉语言（黑白银配色、Cormorant Garamond + Space Grotesk 字体、粒子背景、自定义光标）的前提下，将全屏翻页模式改为纵向长滚动 + sticky header 导航，以承载大幅扩展的业务内容。

## 二、架构变更

| 维度 | v2（当前） | v3（目标） |
|------|-----------|-----------|
| 导航模式 | 全屏翻页（11 slide） | 纵向长滚动 + sticky header |
| 内容组织 | 每 slide 一屏 | 6 大区平铺，视觉分区 |
| 业务详情 | 独立 slide 跳转 | 同页平铺，交替背景区分 |
| 导航 | 右侧圆点 + 左右点击区 | 顶部 sticky header + 锚点高亮 |

## 三、页面骨架（6 大区）

```
0. Hero（全屏）
1. 关于我们
2. 团队架构
   ├── 碳基指挥官
   └── 硅基军团（6 翻转卡片 + 技术底座）
3. 业务一：AI赋能教育与咨询
   ├── 产品矩阵 sticky bar
   ├── 个人成长服务（思维课 + 社群）
   ├── 个人深度陪跑
   ├── 企业内训
   ├── 10天驻场陪跑
   └── 深度咨询
4. 业务二：AIGC创意内容工厂
5. 业务三：大模型算力供应
6. 为什么选择无限涌动（对比表 × 2）
7. 联系我们 + Footer
```

## 四、视觉系统（沿用 v2）

- **配色**：纯黑 `#000` + 近黑 `#0a0a0a`（交替分区）+ 白 `#fff` + 银 `#999` → `#333`
- **字体**：Cormorant Garamond（衬线标题）、Space Grotesk（无衬线正文）、Noto Sans SC（中文）、Noto Serif SC（中文衬线）
- **特效**：粒子背景、自定义光标、鼠标光晕、胶片颗粒层、Intersection Observer 滚动动画
- **交替背景**：大区之间用 `#000` ↔ `#0a0a0a` 交替，形成视觉「翻页感」

## 五、各区域设计详案

### 0. Hero（100vh）

- 不对称分栏：左 55% 文字（SURGE 超大号 ~140px + 中文名 + slogan + CTA），右 45% 留白/海报占位
- 底部居中：向下滚动指示器（浮动箭头动画）
- CTA「开始探索 ↓」平滑滚动到「关于我们」

### 1. 关于我们

- 一行大字横跨：「碳基的理解 × 硅基的执行」
- 两段正文左对齐，max-width 720px，右侧大片留白
- 底部细线分隔

### 2. 团队架构

- **碳基指挥官**：左大头像（视差保留），右姓名/角色/背景
- **硅基军团**：3×2 翻转卡片 + 底部技术底座条。桌面 hover 翻转，移动端点击翻转
- 技术底座文案更新为 OpenClaw 新描述

### 3. 业务一：AI教育与咨询（内容最深的区）

- **产品矩阵 sticky bar**：进入区域后吸附在主 header 下方，4 产品 + 价格横向排列，离开区域消失
- **个人成长服务**：左右分栏 — 左（思维课 ¥299 + 价值简述），右（社群 ¥199/月 + 5 行卡片式栏目表）
- **个人深度陪跑**：左（首月交付清单 — 竖向时间轴），右（月度服务 + 配套权益卡片）
- **企业内训**：简短，3-4 句话
- **10 天驻场陪跑**：横向步骤时间线（Step 0-10）+ 图标
- **深度咨询**：简短卡片收尾
- 会员权益、老带新、答疑机制等辅助信息用小字号灰色呈现
- 四段交替背景区分

### 4. 业务二：AIGC 内容工厂

- 标题 + 副标题
- 三列服务范围卡片
- 横向 3 步生产流程（产品采集 → AI生成 → Skill固化）
- 两阶段服务流程（左右分栏）：阶段一 Demo→Skill ¥3K~¥10K，阶段二批量生成 ¥500/分钟
- 底部工具链 + 核心优势灰色条

### 5. 业务三：大模型算力

- 简短，半屏高度
- 三列图标+文字

### 6. 对比表（两个独立表）

- 表一：业务一维度对比（6 行）— 放在业务一底部
- 表二：业务二维度对比（4 行）— 放在业务二底部
- 逐行 Intersection Observer 动画保留

### 7. 联系我们 + Footer

- 三行联系方式（邮箱/电话/所在地）
- Slogan：「让一个人，拥有一支军团。」
- Footer：© 2026 无限涌动 SURGE

## 六、交互规范

| 元素 | 行为 |
|------|------|
| Sticky header | 滚动超过 Hero 后出现，`backdrop-filter: blur(20px)` + 半透明黑背景 |
| 导航高亮 | Intersection Observer 监测当前区域，对应链接高亮 |
| 产品矩阵 bar | 进入业务一区域出现，滚动出区域消失 |
| 滚动动画 | `opacity 0→1, translateY 30px→0`，0.6s cubic-bezier |
| 翻转卡片 | 桌面 hover 翻转，移动端点击切换 `.flipped` class |
| CEO 头像视差 | 鼠标相对卡片中心偏移，`rotateY/X` 微动 |
| 对比表行 | 每行进入视口时触发 fade-in |
| 粒子背景 | Canvas 全页面，50 粒子，保留 v2 逻辑 |
| 自定义光标 | 保留 v2 圆形光标 + 中心点 + hover 放大 |
| 胶片颗粒 | 保留 SVG overlay |
| 移动端响应 | ≤768px：网格→单列，字号缩小，header→汉堡菜单 |
